# Running the Project Locally (Development Mode)

Use this when you are writing code and want to see changes instantly.

### 1. Open the project in VS Code

Open the folder containing the portfolio project.

### 2. Open the terminal

Press:

```
Ctrl + `
```

### 3. Verify your folder path

Your terminal path should end with:

```
tushar-portfolio
```

If it ends in something like:

```
Tushar Codes
```

run:

```bash
cd tushar-portfolio
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open the site

You will see a link like:

```
http://localhost:5173/
```

Ctrl + Click the link to open the site in the browser.

The page will automatically refresh whenever you save changes.

# Building the Project for Deployment

Use this when you are finished editing and want to upload the changes to the internet.

### 1. Open the terminal

Make sure you are inside the project folder.

### 2. Run the build command

```bash
npm run build
```

Wait for the process to complete.

### 3. Locate the build output

After the build finishes, you will see a folder named:

```
dist
```

This folder contains the optimized production version of the website.

---

# Deploying to Netlify (Drag & Drop Method)

1. Open **Netlify** in your browser.
2. Go to the **Deploy section**.
3. Open your computer's **File Explorer**.
4. Locate the `dist` folder inside the project.
5. Drag the `dist` folder into the Netlify deployment area.

Netlify will automatically deploy the website.
